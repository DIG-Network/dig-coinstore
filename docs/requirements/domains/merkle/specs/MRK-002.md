# MRK-002: Memoized Empty Hashes

> **Authoritative requirement:** [MRK-002](../NORMATIVE.md#MRK-002)
> **Verification:** [VERIFICATION.md](../VERIFICATION.md)
> **Tracking:** [TRACKING.yaml](../TRACKING.yaml)

---

## Summary

A sparse Merkle tree with 256 levels requires frequent access to "empty subtree" hashes -- the hash of a completely empty subtree at each level. Naively computing these recursively on every access is expensive and redundant since they are deterministic constants. This requirement mandates a pre-computed array of 257 empty hashes (one per level, from leaf level 256 down to root level 0), stored in a `OnceLock` for thread-safe lazy initialization and O(1) lookup.

The array has 257 entries because there are 257 distinct levels in a 256-level tree: level 256 (leaf) through level 0 (root).

---

## Specification

### Data Structure

```rust
use std::sync::OnceLock;

static EMPTY_HASHES: OnceLock<[Bytes32; 257]> = OnceLock::new();

/// Returns the pre-computed empty hash at the given tree level.
/// Level 256 = leaf (empty leaf hash), Level 0 = root of an entirely empty tree.
pub fn empty_hash(level: usize) -> Bytes32;
```

### Computation

```
empty_hashes[256] = hash(EMPTY_LEAF)       // or a defined zero/null leaf sentinel
empty_hashes[n]   = hash(empty_hashes[n+1] || empty_hashes[n+1])   for n in 255..=0
```

Where `||` denotes concatenation and `hash` is the tree's hash function (SHA-256).

### Behavior

1. `EMPTY_HASHES` MUST be initialized at most once, on first access, using `OnceLock::get_or_init`.
2. The array MUST contain exactly 257 entries indexed from 0 to 256.
3. `empty_hash(level)` MUST return `EMPTY_HASHES[level]` in O(1) time (direct array index).
4. The leaf-level empty hash (index 256) MUST be the hash of the designated empty leaf sentinel.
5. Each non-leaf empty hash (index n, 0 <= n <= 255) MUST equal `hash(empty_hashes[n+1] || empty_hashes[n+1])`.
6. The implementation MUST NOT perform recursive computation at query time.

---

## Acceptance Criteria

- [ ] `EMPTY_HASHES` is a `OnceLock<[Bytes32; 257]>` static.
- [ ] Initialization computes all 257 values bottom-up in a single pass.
- [ ] `empty_hash(level)` returns the correct value in O(1).
- [ ] `empty_hash(256)` equals the leaf-level empty sentinel hash.
- [ ] `empty_hash(0)` equals the root hash of an entirely empty 256-level tree.
- [ ] Values match those produced by naive recursive computation.
- [ ] Thread-safe: concurrent first-access from multiple threads initializes exactly once.

---

## Implementation Notes

- **Primary file:** `src/merkle/empty_hashes.rs`
- **Dependencies:** None (self-contained utility)
- **Edge cases:**
  - `empty_hash(level)` with `level > 256` SHOULD panic or return an error (out of bounds).
  - The empty leaf sentinel must be consistent across all tree operations.
- **Performance:** One-time initialization cost of 257 hash computations. All subsequent lookups are a single array index.
